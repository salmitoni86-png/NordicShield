# Nordic Shield

The North Will Not Fall.

Nordic Shield is a modern grand strategy and military simulation game focused on the defense of the Nordic region.

Players command military forces across Finland, Sweden, Norway, Denmark, Iceland, and surrounding strategic areas while managing military readiness, intelligence, logistics, and wartime operations.

## Features

- Large-scale strategic map
- Dynamic military campaigns
- Replay system
- Fog of war
- Air, land, naval, and cyber warfare
- Story-driven campaigns
- Sandbox mode
- Mod support (planned)
- Multiplayer (planned)
